# intermediate-elective

## General information

This repository contains the intermediate elective which aims to explore aquatic invertebrates in North Campus Open Space.

To work with the code in this repository, you will need the following packages:

```
library(tidyverse)
library(here)
library(janitor)
library(snakecase)
library(scales)
library(readxl)
library(patchwork)
library(ggtext)
```
